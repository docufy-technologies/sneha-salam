# Sneha Salam

An online portfolio for Sneha Salam, built to present the person first: an operations-minded professional with a strong focus on communication, structure, execution, and people-centered leadership.

## What This Site Shows

- A clear introduction to who Sneha is and how she works
- A practical snapshot of experience across operations, HR, documentation, public relations, and team coordination
- A concise view of strengths such as workflow design, stakeholder management, and cross-functional execution
- A direct path to connect through consultation or email

## Site Flow

- Home: a high-level introduction, impact summary, and key capabilities
- About: a more personal look at values, strengths, and working style
- Experience: professional, volunteering, and academic milestones

## Run Locally

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```
